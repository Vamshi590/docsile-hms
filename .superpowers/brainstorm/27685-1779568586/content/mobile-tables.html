Tables on Mobile

Your app has many data tables (patients, inpatients, labs, billing, insurance). Narrow phone screens can't show 6–8 columns at once. Which treatment do you prefer?

A

Horizontal Scroll

Table stays as-is but scrolls horizontally. Simple to implement. All columns preserved — user swipes to see more. Works well on tablet.

Card Stack (Stacked Rows)

Each row becomes a card on mobile. Key fields shown prominently, secondary info below. Tappable cards — great for touch. More vertical scrolling but no horizontal swipe confusion.
